回退到上一个版本, 上两个版本， 上100个版本 (HEAD^ , HEAD^^, HEAD~100)
git reset --hard HEAD^

回退到某一个版本
git reset --hard <id>

把文件从暂存区重新放入到工作区
git reset HEAD <file>

工作区的撤销修改 (回到最近一次git commit 或 git add 时的状态)
git checkout -- <file>

查看操作日志
git reflog

查看工作区和版本库的区别
git diff HEAD -- <file>


删除文件
rm <file>
git rm <file>

git branch -D dev
